kus is a terminal UI for kuboard.

# Installation

## download


## install
`sh install.sh`

# Description

Configuration file path `~/.kus/config.yaml`

Example:
```yaml
kuboard:
    host: kuboard.xxx.net
    username: Bob
    password: pass
    clusters:
        - DEV
selected:
    cluster: DEV
    namespace: dev
```
